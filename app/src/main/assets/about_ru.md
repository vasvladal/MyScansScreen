# Сканер документов, Обрезка и Создание PDF - Руководство пользователя

## Обзор

Сканер документов - это приложение для сканирования документов на Android, которое позволяет захватывать, улучшать, организовывать и экспортировать документы с помощью камеры вашего смартфона. Приложение имеет расширенную коррекцию перспективы, автоматическое обнаружение документов, инструменты управления изображениями и возможности создания PDF.

## Основные функции

- **Сканирование документов**: Захватывайте документы с помощью камеры вашего устройства
- **Коррекция перспективы**: Автоматическое и ручное определение границ документов
- **Улучшение изображений**: Улучшайте четкость и читаемость документов
- **Управление документами**: Организуйте сканы в виде сетки или списка
- **Создание PDF**: Объединяйте несколько сканов в PDF-документы
- **Поддержка многих языков**: Доступно на английском, русском, украинском и румынском языках
- **Опции общего доступа**: Экспортируйте сканы через различные приложения и сервисы

## Начало работы

### Установка
1. Загрузите и установите приложение из RuStore
2. Предоставьте необходимые разрешения при запросе:
   - Доступ к камере (для захвата документов)
   - Доступ к фото/медиа (для выбора изображений из галереи)
   - Доступ к хранилищу (для сохранения сканированных документов)

### Первый запуск
1. Приложение открывается на экране "Мои сканы", показывая ваши существующие документы
2. Используйте меню в правом верхнем углу для доступа к настройкам языка и информации о приложении
3. Нажмите плавающую кнопку действия, чтобы начать сканирование

## Как использовать приложение

### 1. Захват документов

#### Использование камеры:
1. Нажмите кнопку камеры на главном экране
2. Расположите документ в видоискателе камеры
3. Сделайте снимок, когда документ правильно обрамлен
4. Приложение автоматически обнаружит и исправит перспективу

#### Из галереи:
1. Нажмите кнопку галереи на главном экране
2. Выберите существующее изображение с вашего устройства
3. Приложение обработает изображение для извлечения документа

### 2. Опции обработки

После захвата изображения вы можете выбрать способ его обработки:

#### Автоматическая коррекция (Рекомендуется):
- Нажмите "Авто коррекция" для интеллектуального обнаружения перспективы
- Приложение автоматически идентифицирует границы документов
- Лучше всего подходит для стандартных документов с четкими краями

#### Ручная корректировка перспективы:
- Нажмите "Коррекция перспективы" для ручного контроля
- Перетащите четыре угловые точки, чтобы точно определить границы документов
- Нажмите "Применить" для обработки изображения

#### Базовое обрезание:
- Используйте "Обрезать и улучшить" для простого прямоугольного обрезания
- Отрегулируйте рамку обрезания по необходимости
- Примените базовое улучшение изображения

### 3. Управление вашими сканами

#### Просмотр сканов:
- Все сканы появляются на главном экране "Мои сканы"
- Нажмите любой скан, чтобы просмотреть его на весь экран
- Листайте изображения в режиме полного экрана

#### Организация сканов:
- Получите доступ к виду сетки или списка через меню в правом верхнем углу
- В виде сетки: Долгое нажатие и перетаскивание для изменения порядка сканов
- В виде списка: Используйте ручки перетаскивания для перестановки элементов
- Нажмите значок обновления, чтобы сбросить до первоначального порядка

#### Выбор нескольких сканов:
- Войдите в режим выбора, нажав FAB или сделав долгое нажатие на скан
- Выберите несколько сканов для групповых операций
- Используйте панель выбора внизу для управления выбранными элементами

### 4. Расширенные функции

#### Создание PDF-документов:
1. Выберите несколько сканов в режиме выбора
2. Нажмите значок PDF в верхней панели приложения
3. Подтвердите создание в диалоговом окне
4. PDF будет сохранен в папке ваших документов

#### Общий доступ к сканам:
1. Выберите один или несколько сканов
2. Нажмите значок общего доступа в верхней панели приложения
3. Выберите предпочтительный метод общего доступа
4. Сканами можно делиться как изображениями или PDF-файлами

#### Удаление сканов:
- Индивидуальное: Откройте скан и используйте значок удаления
- Несколько: Выберите сканы и используйте значок удаления
- Подтвердите удаление в появившемся диалоговом окне

### 5. Настройки и предпочтения

#### Выбор языка:
1. Нажмите меню с тремя точками в верхней панели приложения
2. Выберите "Язык"
3. Выберите предпочтительный язык из списка
4. Приложение перезапустится с новым языком

#### Информация о приложении:
- Получите доступ к "О приложении" из меню с тремя точками
- Просмотрите информацию о версии и описание приложения
- Узнайте о функциях и возможностях приложения

## Советы для лучших результатов

### Качество захвата:
- Обеспечьте хорошее освещение при захвате документов
- Расположите документ на контрастном фоне
- Держите камеру параллельно поверхности документа
- Избегайте падения теней на документ

### Советы по обработке:
- Используйте автоматическую коррекцию для стандартных документов
- Ручная коррекция лучше всего работает для сложных фонов
- Для изогнутых документов используйте несколько точек ручной регулировки

### Организация:
- Называйте ваши сканы содержательно для легкой идентификации
- Используйте вид сетки для визуального просмотра
- Используйте вид списка, когда у вас много сканов для управления

## Решение проблем

### Распространенные проблемы:

**Камера не работает:**
- Проверьте разрешения камеры в настройках приложения
- Убедитесь, что никакое другое приложение не использует камеру
- Перезапустите приложение, если проблемы продолжаются

**Плохие результаты обнаружения:**
- Улучшите условия освещения
- Попробуйте ручную коррекцию перспективы
- Сделайте снимок документа под более прямым углом

**Проблемы с хранилищем:**
- Проверьте разрешения хранилища
- Убедитесь, что доступно достаточно места в хранилище
- Очистите кеш приложения при необходимости

### Поддержка:
Для дополнительной поддержки проверьте раздел "О приложении" для контактной информации или ссылок на документацию.
