# Сканер документів, Обрізка та Створення PDF - Керівництво користувача

## Огляд

Сканер документів - це додаток для сканування документів на Android, який дозволяє захоплювати, покращувати, організовувати та експортувати документи за допомогою камери вашого смартфона. Додаток має розширену корекцію перспективи, автоматичне виявлення документів, інструменти керування зображеннями та можливості створення PDF.

## Основні функції

- **Сканування документів**: Захоплюйте документи за допомогою камери вашого пристрою
- **Корекція перспективи**: Автоматичне та ручне визначення меж документів
- **Покращення зображень**: Покращуйте чіткість та читабельність документів
- **Управління документами**: Організовуйте скани у вигляді сітки або списку
- **Створення PDF**: Об'єднуйте кілька сканів у PDF-документи
- **Підтримка багатьох мов**: Доступно англійською, російською, українською та румунською мовами
- **Опції спільного доступу**: Експортуйте скани через різні програми та сервіси

## Початок роботи

### Встановлення
1. Завантажте та встановіть додаток з RuStore
2. Надайте необхідні дозволи при запиті:
   - Доступ до камери (для захоплення документів)
   - Доступ до фото/медіа (для вибору зображень з галереї)
   - Доступ до сховища (для збереження сканованих документів)

### Перший запуск
1. Додаток відкривається на екрані "Мої скани", показуючи ваші існуючі документи
2. Використовуйте меню у верхньому правому куті для доступу до налаштувань мови та інформації про додаток
3. Натисніть плаваючу кнопку дії, щоб розпочати сканування

## Як використовувати додаток

### 1. Захоплення документів

#### Використання камери:
1. Натисніть кнопку камери на головному екрані
2. Розмістіть документ у відошукачі камери
3. Зробіть знімок, коли документ правильно обрамлений
4. Додаток автоматично виявить та виправить перспективу

#### З галереї:
1. Натисніть кнопку галереї на головному екрані
2. Виберіть існуюче зображення з вашого пристрою
3. Додаток обробить зображення для витягування документа

### 2. Опції обробки

Після захоплення зображення ви можете вибрати спосіб його обробки:

#### Автоматична корекція (Рекомендовано):
- Натисніть "Авто корекція" для розумного виявлення перспективи
- Додаток автоматично ідентифікує межі документів
- Найкраще підходить для стандартних документів з чіткими краями

#### Ручне коригування перспективи:
- Натисніть "Корекція перспективи" для ручного контролю
- Перетягніть чотири кутові точки, щоб точно визначити межі документів
- Натисніть "Застосувати" для обробки зображення

#### Базове обрізання:
- Використовуйте "Обрізати та покращити" для простого прямокутного обрізання
- Відрегулюйте рамку обрізання за потребою
- Застосуйте базове покращення зображення

### 3. Керування вашими сканами

#### Перегляд сканів:
- Усі скани з'являються на головному екрані "Мої скани"
- Натисніть будь-який скан, щоб переглянути його на весь екран
- Гортайте зображення в режимі повного екрану

#### Організація сканів:
- Отримайте доступ до вигляду сітки або списку через меню у верхньому правому куті
- У вигляді сітки: Довге натискання та перетягування для зміни порядку сканів
- У вигляді списку: Використовуйте ручки перетягування для перестановки елементів
- Натисніть значок оновлення, щоб скинути до початкового порядку

#### Вибір кількох сканів:
- Увійдіть у режим вибору, натиснувши FAB або зробивши довге натискання на скан
- Виберіть кілька сканів для групових операцій
- Використовуйте панель вибору внизу для керування вибраними елементами

### 4. Розширені функції

#### Створення PDF-документів:
1. Виберіть кілька сканів у режимі вибору
2. Натисніть значок PDF у верхній панелі додатку
3. Підтвердіть створення в діалоговому вікні
4. PDF буде збережено у папці ваших документів

#### Спільний доступ до сканів:
1. Виберіть один або кілька сканів
2. Натисніть значок спільного доступу у верхній панелі додатку
3. Виберіть бажаний метод спільного доступу
4. Сканами можна ділитися як зображеннями або PDF-файлами

#### Видалення сканів:
- Індивідуальне: Відкрийте скан і використовуйте значок видалення
- Кілька: Виберіть скани та використовуйте значок видалення
- Підтвердіть видалення у діалоговому вікні, що з'явиться

### 5. Налаштування та уподобання

#### Вибір мови:
1. Натисніть меню з трьома крапками у верхній панелі додатку
2. Виберіть "Мова"
3. Виберіть бажану мову зі списку
4. Додаток перезапуститься з новою мовою

#### Інформація про додаток:
- Отримайте доступ до "Про додаток" з меню з трьома крапками
- Перегляньте інформацію про версію та опис додатка
- Дізнайтеся про функції та можливості додатка

## Поради для найкращих результатів

### Якість захоплення:
- Забезпечте хороше освітлення під час захоплення документів
- Розмістіть документ на контрастному фоні
- Тримайте камеру паралельно поверхні документа
- Уникайте падіння тіней на документ

### Поради з обробки:
- Використовуйте автоматичну корекцію для стандартних документів
- Ручна корекція найкраще працює для складних фонів
- Для вигнутих документів використовуйте кілька точок ручного коригування

### Організація:
- Називайте ваші скани змістовно для легкої ідентифікації
- Використовуйте вигляд сітки для візуального перегляду
- Використовуйте вигляд списку, коли у вас багато сканів для керування

## Вирішення проблем

### Поширені проблеми:

**Камера не працює:**
- Перевірте дозволи камери в налаштуваннях додатка
- Переконайтеся, що жоден інший додаток не використовує камеру
- Перезапустіть додаток, якщо проблеми продовжуються

**Погані результати виявлення:**
- Покращіть умови освітлення
- Спробуйте ручну корекцію перспективи
- Зробіть знімок документа під більш прямим кутом

**Проблеми зі сховищем:**
- Перевірте дозволи сховища
- Переконайтеся, що доступно достатньо місця у сховищі
- Очистіть кеш додатка за потреби

### Підтримка:
Для додаткової підтримки перевірте розділ "Про додаток" для контактної інформації або посилань на документацію.
